# 个人资源小站

这是一个简单的个人资源管理网站，可以创建、编辑和删除资源帖子。

## 功能特点

- 管理员登录功能
- 帖子创建、编辑和删除
- 帖子内容中的URL自动转换为可点击链接
- 响应式设计，适配不同设备
- 纯文本界面，无图标

## 使用方法

1. 克隆或下载本项目
2. 打开 `index.html` 文件
3. 点击右上角"管理员登录"按钮登录
4. 开始管理您的资源帖子

## 技术栈

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap
- localStorage 存储数据
